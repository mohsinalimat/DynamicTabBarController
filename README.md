# DynamicTabBarController
An improved UITabBarController that is dynamic and has a scroll indicator line

### Installation via CocoaPods

```ruby
pod 'DynamicTabBarController'
```

### Requirements

iOS 9.0+
Swift 4
XCode 9.0+
